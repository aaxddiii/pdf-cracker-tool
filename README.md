PDF Cracker Tool Using Python
A complete educational tool that demonstrates how dictionary attacks + brute force attacks work on password-protected PDF files using:
1. pikepdf
2. multithreading
3. wordlist processing
4. generators
5. itertools
6. argparse CLI interface
7. progress bars (tqdm)

Features
✅ Dictionary-based password testing
✅ Brute-force password generation
✅ Multi-threading for faster execution
✅ tqdm progress bar
✅ Exception handling
✅ Command-line interface
✅ Clean, modular, readable code
✅ Fully documented & beginner-friendly
